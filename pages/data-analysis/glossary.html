Glossary

  1. Overview of data analysis

    • KPI (Key performance indicators): measure which indicates a company’s success vs. a set of targets, objectives, or industry peers. **
    • ROI (Return of investment): the ratio of net profit over the total cost of investment.
    • Machine learning: a branch of artificial intelligence (AI) and computer science which focuses on the use of data and algorithms to imitate the way that humans learn, gradually improving its accuracy.
    • Neural network: a series of algorithms that endeavors to recognize underlying relationships in a set of data through a process that mimics the way the human brain operates.
    • Decision tree: non-parametric supervised learning method used for classification and regression.
    • Regression: a method for understanding the relationship between independent variables or features and a dependent variable or outcome.

  2. Roles in data

    • Data wrangling: process of converting raw data into a usable form. It may also be called data munging or data remediation. You’ll typically go through the data wrangling process prior to conducing any data analysis in order to ensure your data is reliable and complete.
    • Data warehouse: system used for the analysis and reporting of structured and semi-structured data from multiple sources.
    • Data lake: centralized repository designed to store, process, and secure large amounts of structured, semi-structured, and unstructured data.
    • Exploratory data analysis (EDA): analysis technique used by data scientists that refers to the critical process of performing initial investigations on data so as to discover patterns, to sport anomalies, to test hypothesis and to check assumptions with the help of summary statistics and graphical representations.
    • Forecasting model: statistical tool designed to predict future trends and outcomes based on historical data. It involves analyzing past patterns and trends to make informed predictions about future events, sales, demand, or inventory levels.
    • Deep learning: machine learning technique that teaches computers to do what comes naturally to humans: learn by example.

  3. Tasks of a data analyst

    • Cognitive Services: a set of machine learning algorithms that Microsoft has developed to solve problems in the field of Artificial Intelligence (AI).
    • Data silo: collection of data held by one group that is not easily or fully accessible by other groups in the same organization.
